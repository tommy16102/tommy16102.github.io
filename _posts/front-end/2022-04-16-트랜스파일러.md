## 트랜스파일러(transpiler)  
- 같은 언어로 작성된 코드를 문법적으로 변환을 해주는 도구.   
- 예시  
  - Typescript => Javascript  
  - scss => css (SASS)  
  - Javascript(ES6) => Javascript(ES5) (Babel)  
  
### 컴파일러(compiler)  
- 코드를 다른 언어로 변환.  
- 예시) C => assembly.
